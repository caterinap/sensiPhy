Influential clade detection - Phylogenetic Linear Regression

Usage

clade_phylolm(formula, data, phy, model = "lambda", track = TRUE, clade.col, n.species = 5, ...)

Arguments

formula
The model formula
data
Data frame containing species traits with row names matching tips in phy.
phy
A phylogeny (class 'phylo') matching data.
model
The phylogenetic model to use (see Details). Default is lambda.
track
Print a report tracking function progress (default = TRUE)
clade.col
The name of a column in the provided data frame with clades specification (a character vector with clade names).
n.species
Minimum number of species in the clade in order to include this clade in the leave-one-out deletion analyis. Default is 5.
...
Further arguments to be passed to phylolm

Value

The function clade_phylolm returns a list with the following components:

formula: The formula

full.model.estimates: Coefficients, aic and the optimised value of the phylogenetic parameter (e.g. lambda) for the full model without deleted species.

clade.model.estimates: A data frame with all simulation estimates. Each row represents a deleted clade. Reported are the calculated regression intercept (intercept), difference between simulation intercept and full model intercept (DFintercept), the percentage change in intercept compared to the full model (intercept.perc) and intercept p-value (pval.intercept). All of these are also reported for the regression slope (DFslope etc.). Additonally, model aic value (AIC) and the optimised value (optpar) of the phylogenetic parameter (e.g. kappa or lambda, depends on phylogeneticmodel used) are reported.

data: Original full dataset.

errors: Clades where deletion resulted in errors.

Description

Estimate the impact on model estimates phylogenetic linear regression after removing clades from the analysis.

Details

This function sequentially removes one clade at a time, fits a phylogenetic linear regression model using phylolm and stores the results. The impact of of a specific clade on model estimates is calculated by the comparison between the full model (with all species) and the model without the species belonging to a clade.

All phylogenetic models from phylolm can be used, i.e. BM, OUfixedRoot, OUrandomRoot, lambda, kappa, delta, EB and trend. See ?phylolm for details.

clade_phylolm detects influential clades based on difference in intercept and/or slope when removing a given clade compared to the full model including all species.

Currently, this function can only implement simple linear models (i.e. $y = a + bx$). In the future we will implement more complex models.

Output can be visualised using sensi_plot.

Intraspecific variability - Phylogenetic Linear Regression

Usage

intra_phylolm(formula, data, phy, vari.resp = NULL, vari.pred = NULL, minmax = FALSE, nintra = 2, distrib = "uniform", model = "lambda", track = TRUE, ...)

Arguments

formula
The model formula: response~predictor. If minmax=TRUE the formula should point to the columns containing the minimum values of the response and/or predictor variables i.e. min.response~min.predictor.
data
Data frame containing species traits with species as row names.
phy
A phylogeny (class 'phylo', see ?ape).
vari.resp
Name of the column containing the standard error, the standard deviation of the response variable or its maximum value (if minmax=TRUE). When information is not available for one taxon, the value can be 0 or NA.
vari.pred
Name of the column containing the standard error, the standard deviation of the predictor variable or its maximum value (if minmax=TRUE). When information is not available for one taxon, the value can be 0 or NA
minmax
logical; if TRUE, generates a value in the interval [min,max]. If TRUE, please select columns containing minimum values in the formula and columns containig maximum values in vari.resp and/or vari.pred (default = FALSE).
nintra
Number of times to repeat the analysis generating a random value for response and/or predictor variables. If NULL, nintra = 2
distrib
A character string indicating which distribution to use to generate a random value for the response and/or predictor variables.Default is uniform distribution: "uniform" (runif) normal distribution is "normal" (function rnorm). Warning: normal distribution can be used oly if vari.pred is the standard deviation of the mean. If minmax=T, only "uniform" distribution is available.
model
The phylogenetic model to use (see Details). Default is lambda.
track
Print a report tracking function progress (default = TRUE)
...
Further arguments to be passed to phylolm

Value

The function intra_phylolm returns a list with the following components:

formula: The formula

data: Original full dataset

model_results: Coefficients, aic and the optimised value of the phylogenetic parameter (e.g. lambda) for each regression with a different phylogenetic tree.

N.obs: Size of the dataset after matching it with tree tips and removing NA's.

stats: Statistics for model parameters. sd_tree is the standard deviation due to phylogenetic uncertainty.

Description

Performs Phylogenetic linear regression evaluating intraspecific variability.

Details

This function fits a phylogenetic linear regression model using phylolm. The regression is repeated nintra times. At each iteration the functions generates for each row in the dataset a random value in the interval [predictor-vari.pred,predictor+vari.pred] and a random value in the interval [response-vari.resp,response+vari.resp]. If minmax=TRUE the value is randomly generated in the interval [min,max]. If you log-transform your predictor and/or response variables, make sure that vari.pred and/or vari.resp are in a log-scale too.

All phylogenetic models from phylolm can be used, i.e. BM, OUfixedRoot, OUrandomRoot, lambda, kappa, delta, EB and trend. See ?phylolm for details.

Currently, this function can only implement simple linear models (i.e. $trait~ predictor$). In the future we will implement more complex models.

Output can be visualised using sensi_plot.

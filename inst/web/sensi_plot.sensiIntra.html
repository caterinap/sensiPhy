Graphical diagnostics for class 'sensiIntra'

Usage

"sensi_plot"(x, graphs = "all", ...)

Arguments

x
output from tree_phylolm, tree_phyloglm, intra_phylolm or intra_phyloglm
graphs
choose which graph should be printed in the output ("all", 1, 2 or 3)
...
further arguments to methods

Description

plot_tree.intra_phylolm Plot results from tree_phylolm, intra_phylolm and intra_phyloglm

Details

For 'x' from tree_phylolm, tree_phyloglm, intra_phylolm or intra_phyloglm:

Graphs 1 and 2: Distribution of estimated slopes and intercepts for each tree (for tree_phylolm) or value generated within a given interval (intra_phylolm) Red vertical line represents the mean slope or intercept for all models.

Graph 3: Scatterplot with mean regression (black line) and standard deviation of the regression (blue dotted lines).
